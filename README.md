### 2.1 创建套接字
* 2.1.1 协议栈的内部结构<br/>
  **协议栈**是指网络中各层协议的总和，本质是**操作系统内部的网络控制软件**，也叫**协议驱动**或者**TCP/IP驱动**，其形象的反映了一个网络中文件传输的过程：由上层协议到底层协议，再由底层协议到上层协议。<br/>
  协议栈内部主要分为两层：**TCP协议和UDP协议**作为一层，其下面还有**IP协议**作为另一层；<br/>
  整个系统分层结构为:

  **Socket库**
